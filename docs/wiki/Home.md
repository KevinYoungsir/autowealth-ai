## 项目简介

AutoWealth AI 是一款基于多智能体技术的开源投资分析引擎。

### 核心特性

- **多智能体协作** - 3个专业AI智能体协同工作
- **多维度分析** - 技术指标、基本面、市场情绪全覆盖
- **本地优先** - 支持本地LLM部署，保护数据隐私
- **批量处理** - 一键分析多只股票，快速筛选
- **可视化界面** - Streamlit交互界面

### 技术架构

```
AutoWealth Engine
├── Technical Analyst Agent
├── Fundamental Analyst Agent
├── Sentiment Analyst Agent
└── Agent Coordinator (Weighted Voting)
```
