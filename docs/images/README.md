## 截图规范

1. **格式**: PNG（透明背景或白色背景均可）
2. **分辨率**: 至少 1280 x 800 px，推荐 2x 分辨率用于 Retina 屏幕
3. **语言**: 界面语言为中文
4. **数据**: 使用模拟数据，不展示真实敏感信息
5. **风格**: 保持界面简洁、专业，避免过多装饰元素
6. **文件大小**: 单张截图不超过 500KB（压缩后）

## 添加截图
